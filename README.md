# Camila Premium Invitation

Plantilla estática premium para invitación digital de XV años.

## Cómo probar

Abre `index.html` en el navegador o súbelo a GitHub Pages / Cloudflare Pages.

## Links de prueba para invitados

- `index.html?guest=familia-gonzalez`
- `index.html?guest=familia-rodriguez`
- `index.html?guest=mariana`
- `index.html?guest=valeria`

## Qué editar para otro cliente

- `data/event.js`: datos del evento, fecha, lugares, WhatsApp, música, textos.
- `data/guests.js`: invitados y pases.
- `assets/photos/`: fotos.
- `assets/music/song.mp3`: canción.

## Recomendación

Optimizar fotos a WebP antes del deploy final.
